Este projeto foi desenvolvido como parte de uma atividade prática da disciplina de Programação Web. O objetivo é aplicar conceitos básicos de HTML e CSS para criar uma página simples de perfil profissional, organizando informações como apresentação pessoal, habilidades e formas de contato. Durante o desenvolvimento, foram utilizados recursos de estilização em CSS para melhorar a aparência da página, como cores, fontes, alinhamento de texto e controle de imagens. O trabalho busca demonstrar a compreensão inicial dos fundamentos de desenvolvimento web e a capacidade de estruturar e estilizar uma página de forma clara e organizada.
